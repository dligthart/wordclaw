Core Capabilities

WordClaw helps agents work on governed content, assets, and paid-access flows without losing human oversight or runtime safety.

🔌

REST + Reactive MCP

Expose the core runtime through deterministic REST contracts and remote MCP sessions that can subscribe to review, content, and integration events in real time.

🧭

Capability Manifest

Let agents inspect transports, actor profiles, enabled modules, task recipes, storage providers, and runtime query support before they mutate state.

📊

Content Runtime Queries

Use schema-aware field filters, grouped projections, and lifecycle-aware reads for leaderboard, analytics, and session-style content models.

🖼️

Schema-Aware Assets

Store assets as first-class records with local or S3-compatible backends, signed or entitled delivery, and direct-provider upload flows.

🛡️

Workflow + Approvals

Route content changes through review gates so agents can draft and propose safely while humans retain final control over sensitive transitions.

⚡

Built-in L402 Payments

Run Lightning-gated offers, entitlement purchases, and paid reads as part of the core runtime contract.

🧾

Public Write Lanes

Issue short-lived write tokens for player or session-like actors without exposing the full supervisor API-key surface.

📝

Supervisor Control Plane

A focused oversight UI for approvals, audits, schemas, content inspection, payments, and asset lifecycle management.

🌐

Multi-Tenant Governance

Host multiple projects or organizations on one instance with domain-scoped content, credentials, workflows, and delivery policy boundaries.

Built for Governed Automation

WordClaw is now more than schema CRUD. The live runtime combines target resolution, lifecycle-aware content reads, first-class asset delivery, and reactive agent coordination in one governable control plane.

  • Backend: Node.js, Fastify, Drizzle ORM
  • Database: PostgreSQL with pgvector
  • Content Runtime: schema refs, field-aware queries, grouped projections, public write lanes, and TTL lifecycle archival
  • Asset Layer: local or S3-compatible storage with public, signed, and entitled delivery
  • Control Plane: SvelteKit supervisor UI with asset lifecycle controls
  • Agent Surface: REST, MCP, and remote reactive subscriptions

What ships today?

Schema-aware content and assets, review workflows, L402 paid reads, public write lanes, and reactive MCP sessions are all live on `main`.
