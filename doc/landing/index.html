The Safe Content Runtime for AI Agents

Designed for governed automation. WordClaw combines structured content contracts, dry-run safety, workflow approvals, auditability, and agent-native REST/MCP access in one runtime, with optional semantic search and Lightning-gated access when you need them.

Core Capabilities

WordClaw helps agents work on content without removing human oversight or governance.

🔌

REST + MCP Surfaces

Expose the core runtime through primary agent-facing surfaces with deterministic guidance, simulation hooks, and policy-aware mutations.

⚡

Optional L402 Access

Add Lightning-gated offers and entitlement-based reads when paid machine access is part of the workflow.

🧠

Native Vector RAG

Built-in pgvector support. Upload documents and WordClaw automatically chunks, embeds, and serves them for rapid semantic retrieval by downstream agents.

🛡️

Workflow + Approvals

Route content changes through review gates so agents can draft and propose safely while humans retain final control over sensitive transitions.

🌐

Multi-Tenant Governance

Host multiple projects or organizations on one instance with domain-scoped content, credentials, and workflow boundaries.

📝

Supervisor Control Plane

A focused oversight UI for approvals, audits, schemas, content inspection, and agent-key management.

Built for Governed Automation

WordClaw uses a modern stack designed for high-trust agent operations, not just generic CMS CRUD.

  • Backend: Node.js, Fastify, Drizzle ORM
  • Database: PostgreSQL with pgvector
  • Control Plane: SvelteKit supervisor UI
  • Agent Surface: REST and Model Context Protocol (MCP)
