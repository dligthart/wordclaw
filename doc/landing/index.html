Focused Product Direction

The Safe Content Runtime for AI Agents

Designed for governed automation. WordClaw combines structured content contracts, dry-run safety, workflow approvals, auditability, and agent-native REST/MCP access in one runtime. Agents can discover the live deployment contract first, confirm their current actor, then follow task-specific guidance for content, review, integrations, and paid-content flows.

Core Capabilities

WordClaw helps agents work on content without removing human oversight or governance.

🔌

REST + MCP Surfaces

Expose the core runtime through primary agent-facing surfaces with deterministic guidance, simulation hooks, remote MCP attachment, and policy-aware mutations.

🧭

Capability Manifest

Let agents inspect transports, actor profiles, enabled modules, and task recipes before they choose a surface or mutate state.

⚡

Built-in L402 Payments

Run Lightning-gated offers, entitlement purchases, and paid reads as part of the core runtime contract.

🛡️

Workflow + Approvals

Route content changes through review gates so agents can draft and propose safely while humans retain final control over sensitive transitions.

🧠

Native Vector RAG

Built-in pgvector support. Upload documents and WordClaw automatically chunks, embeds, and serves them for rapid semantic retrieval by downstream agents.

🌐

Multi-Tenant Governance

Host multiple projects or organizations on one instance with domain-scoped content, credentials, and workflow boundaries.

📝

Supervisor Control Plane

A focused oversight UI for approvals, audits, schemas, content inspection, payments, and API key management.

Built for Governed Automation

WordClaw uses a modern stack designed for high-trust agent operations, not just generic CMS CRUD.

  • Backend: Node.js, Fastify, Drizzle ORM
  • Database: PostgreSQL with pgvector
  • Control Plane: SvelteKit supervisor UI
  • Agent Surface: REST and Model Context Protocol (MCP)
